# 爬取文旅部和广电总局新闻信息
### 动因及吐槽
已经30多岁了，可领导却让我每周整理文旅部和广电总局发布的有商业价值的消息。要把网站里的新闻都看个遍，一条一条点开实在是浪费时间。怎么看都像是个实习生的活儿，光劳神没什么实际价值和提升。于是尝试用爬虫将所有的新闻合并到一个页面下，既能一目十行还能顺便学习下爬虫和类的用法。
### 功能

使用xpath进行解析并保存至sqlite数据库
### 使用方法
1、运行main.py
2、生成数据库文件
3、运行web.py,按输出打开浏览器窗口
4、enjoy！
